# QA Lead

## Overview
The QA Lead defines the project's quality strategy, manages testing, and assures that deliverables meet set standards. They coordinate QA team members and work closely with development and release management.

## Key Responsibilities
- Define and own the quality assurance plan
- Manage QA resources and testing efforts
- Coordinate bug tracking and resolution
- Collaborate with developers and release teams

## Typical Interactions
- Developers: For test planning, bug reports, and fixes
- Release Manager: For readiness and deployment checks
- Project Manager: For reporting of QA metrics

# Technical Writer

## Overview
The Technical Writer provides documentation support across the project lifecycle, including process docs, user manuals, release notes, and API guides, ensuring clear and complete deliverables for all audiences.

## Key Responsibilities
- Develop and maintain project documentation
- Collaborate with business analysts, engineers, and QA
- Ensure documentation is user-friendly and up-to-date
- Gather feedback and improve documentation iteratively

## Typical Interactions
- Business Analyst: For process flow details
- Engineers: For technical specifications
- QA Lead: For documentation on releases and test coverage
